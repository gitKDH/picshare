# Theme-Market

SpringBoot로 제작한 
애플리케이션 마켓 프로젝트 입니다.

## 개발 환경

- Java 11
- Server : Apache Tomcat 9.0
- Framework : Springboot, JPA
- Database: MySql
- IDE : IntelliJ

## 사용 기술

### [Back-End]

- JAVA 11
- SpringBoot
- Spring Data JPA
- MySql

### [Front-End]

- Thymeleaf
- HTML/CSS
- JavaScript
- jQuery
